## For AI Agents — Start Here
Read these files at the start of every session:
1. `PROJECT.md` — what we're building and why
2. `STAGES.md` — where we are in the build
3. `CURRENT.md` — what to work on right now
4. `SCHEMA.md` — database structure
5. `DECISIONS.md` — decisions already made (don't re-debate)
6. `BUGS.md` — known issues pending fixes
